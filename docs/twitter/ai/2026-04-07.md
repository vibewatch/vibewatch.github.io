### 1.4 Stanford's "Mirage Effect": Multimodal AI Doesn't Actually See (↑ emerging)

[@heynavtoor](https://x.com/heynavtoor) shared Stanford's "Mirage" paper co-authored by Fei-Fei Li, revealing that frontier multimodal models fabricate visual understanding when no images are provided (135 likes, 79 bookmarks, 14.8K views, [tweet](https://x.com/heynavtoor/status/2041618425366360415)). Key findings: models described nonexistent images in detail over 60% of the time; retained 70–80% of benchmark scores with all images removed; a 3B text-only model outperformed every frontier multimodal model on chest X-ray benchmarks by more than 10%. When hallucinating medical scans, models biased toward the worst diagnoses — heart attacks, melanoma, carcinoma.

![Stanford "Mirage: The Illusion of Visual Understanding" paper abstract — documenting how frontier models fabricate detailed image descriptions for images never provided, with pathology-biased clinical findings](https://pbs.twimg.com/media/HFVJI0Va0AA4xwy.png)

The thread detail revealed the most alarming numbers: GPT-5.1's actual vision ability on MicroVQA dropped from 61.5% to 15.4% after removing questions answerable without seeing the image — three quarters of its "visual understanding" was text pattern matching. Stanford introduces B-Clean as a principled benchmark solution.

**Severity: Critical | Prevalence: High**

## 6. Emerging Signals

### Claude Mythos as Restricted Capability Model
Anthropic's decision to gate Mythos behind a "Cybersecurity Consortium" rather than release it publicly represents a new deployment paradigm: models too capable to release broadly. This contrasts sharply with GLM-5.1's MIT-licensed open release on the same day. The dual-use tension Hassabis described — "these systems can be used for harm as well as good" — is now playing out in concrete access decisions.

### China's GPU Independence Is Real
GLM-5.1 was trained entirely on Huawei Ascend chips, demonstrating that U.S. export controls on Nvidia hardware have failed to prevent competitive AI development. The model achieves ~95% of Opus 4.6 quality at 1/5th the API cost. [@grok](https://x.com/grok) noted: "Sanctions created an independent rival ready to undercut prices worldwide." Zhipu's $6.5B Hong Kong IPO and 500%+ stock jump suggest markets agree.

### AI Security Agents Finding Real CVEs
[@_colemurray](https://x.com/_colemurray)'s waclaude finding arbitrary code execution in HuggingFace's transformers library (CVE-2026-1839) is a concrete proof point for pmarca's thesis. The vulnerability was "literally a one-liner PyTorch `load_weights` call" — the kind of obvious-in-hindsight flaw that persists in widely-used libraries because human auditing can't scale. AI security agents are graduating from research demos to real vulnerability discovery.

### Multimodal AI's Vision Problem
Stanford's Mirage paper isn't just an academic concern — 230 million people ask AI health questions daily. The finding that a text-only model outperforms frontier multimodal models on chest X-ray diagnosis by 10%+ means current benchmarks have been systemically misleading the field. The pathology bias (models hallucinate the worst diagnoses, not benign ones) creates a clinical safety problem of the first order.

### Human Review as Bottleneck
The ChipAgents lecture slide on the "Human Review Bottleneck" — citing ~1,000 LoC/day as the upper bound on human code review capacity, analogous to Amdahl's Law — formalizes a constraint that will define the next phase of agentic AI. Tools that increase the bandwidth of human reviewers (not replace them) will be more valuable than tools that increase AI code output further.
